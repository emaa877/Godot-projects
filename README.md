# Pruebas de Juegos e Interfaces Gráficas en Godot

---

## Descripción

Este repositorio contiene una serie de **pruebas y prototipos desarrollados con el motor de videojuegos**, enfocados en:

- Desarrollo de juegos simples (2D)
- Experimentación con interfaces gráficas (UI)
- Validación de mecánicas básicas de interacción

Godot es un motor open-source utilizado para crear aplicaciones interactivas y videojuegos multiplataforma, con soporte tanto para 2D como 3D. 

---

## Objetivos

- Explorar las capacidades del motor Godot
- Implementar prototipos funcionales de juegos
- Diseñar e integrar interfaces gráficas (menus, HUD, inputs)
- Comprender la arquitectura basada en nodos y escenas

---

## Contenido del Repositorio

- **Juegos de prueba**
  - Mecánicas básicas (movimiento, colisiones, inputs)
  - Escenas simples en 2D 

- **Interfaces gráficas**
  - Menús interactivos
  - Botones y navegación
  - Layouts y organización de UI

- **Experimentos**
  - Pruebas de scripts (GDScript)
  - Integración de señales (signals)
  - Manejo de escenas

---

## Tecnologías Utilizadas

- **Motor:** Godot Engine 
- **Lenguaje:** GDScript 
- **Plataforma:** PC (Windows/Linux)

---

## Aprendizajes

- Uso de la arquitectura basada en **nodos y escenas**
- Manejo de eventos mediante **signals**
- Implementación de lógica de juego básica
- Diseño de interfaces gráficas funcionales

Proyecto en desarrollo / experimental.
